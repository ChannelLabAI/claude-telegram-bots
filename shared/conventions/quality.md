# Quality Conventions

## Citation Rule
- 每個事實必須有來源引用（citation）
- 無來源 = 不可信，不寫入產出

## Wikilink Rule
- 每次提到有 Ocean page 的人、公司、專案，必須建立 `[[wikilink]]`
- 沒有 link 的 mention = broken brain
- 產出的文件必須有 wikilink 接入知識圖譜，不製造孤島

## Pearl 萃取
- 對話中出現洞見時，標記為 Pearl 草稿候選
- 後續由人工或流程決定是否正式收錄至 `Pearl/`

## Seabed 保護
- Seabed 是唯一原始記錄
- 禁止在 Seabed 以外修改對話記錄
- 任何引用 Seabed 內容的地方，只能引用，不能改寫原文
